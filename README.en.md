# 🏴‍☠️ Treasure Hunters

![License](https://img.shields.io/badge/License-MIT-blue.svg)
![Phaser](https://img.shields.io/badge/Phaser-3.80.1-brightgreen.svg)
![Vite](https://img.shields.io/badge/Vite-5.0.0-yellow.svg)

A 2D platformer adventure game about hunting pirate treasures. Built with the Phaser 3 framework and bundled with Vite.

![Gameplay Screenshot](./public/assets/environment/Background/bg_color.png) *(Note: Replace with actual gameplay screenshot)*

</div>

## 🎮 Key Features

- **4 Exploration Levels**: Overcome obstacles from the beach to the pirate ship.
- **Physics System**: Dynamic gravity and collision system with objects.
- **Combat System**: Players can use a sword to defeat enemies.
- **Treasure Collection**: Collect coins and golden chests.
- **Dynamic Obstacles**: Cannon traps, spikes, and patrolling enemies.
- **Smooth Sprite Animations**: Utilizing sprite sheets for characters, enemies, and environments.

## 🚀 Installation Guide

For a complete guide on how to run this project on your local machine, please refer to the [SETUP_GUIDE.en.md](./Doc/SETUP_GUIDE.en.md).

### Prerequisites:
- Node.js (v16 or newer)
- npm or yarn

### Quick Install:
```bash
# Clone the repository
git clone https://github.com/username/treasure-hunters.git
cd treasure-hunters

# Install dependencies
npm install

# Run development server
npm run dev
```
